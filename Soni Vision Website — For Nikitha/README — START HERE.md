### Design Direction
- **Benchmark:** Maloney-Shamie Vision Institute — premium, sophisticated, high-end surgical feel
- **Not:** clinical, corporate, template-y, or busy
- **Physician-first:** The doctors are the product. Lead with people, not services.
- **Trust signals:** 4.9 stars should be front and center, never buried
- **White space:** Signals quality. When in doubt, add space, not content.
- **Typography:** Serif display font for headings (premium feel) + clean sans-serif for body
- **Mobile-first:** 50%+ of medical searches are on mobile

### Technical Context
- The V1 was built with plain HTML/CSS/vanilla JS (no frameworks)
- Deployed on Vercel from GitHub repo `spatel0115/soni-vision-website`
- The site has 30+ pages covering all services, conditions, blog, reviews, referrals
- SEO strategy is documented in the competitive analysis — schema markup, meta tags, and dedicated pages for each IOL type are important to keep regardless of design direction
